# ☕ CaféScript

CaféScript is a coffee-inspired educational platform designed to help people learn programming step by step in a simple, structured, and friendly way.
It focuses on clear explanations, progressive learning paths, and dynamic content loaded from JSON files.


## 🚀 What is this project?

CaféScript is a web-based learning platform where programming courses are organized by technologies.  
Each lesson is dynamically rendered using a custom JavaScript system, making the content easy to scale and maintain.


## 🧠 Technologies covered

- HTML  
- CSS  
- JavaScript  
- Python  
- C++  
- Java  
- C#  
- SQL  


## ⚙️ Features

- Structured learning paths per technology  
- JSON-based lesson system  
- Dynamic content renderer  
- Custom UI with animations and responsive design  
- Support for worksheets and downloadable resources  
- Support for embedded video lessons (YouTube iframe)  


## 🚧 Project status

This project is currently under active development.  
Some courses and lessons may be incomplete or unavailable.


## ☕ Philosophy

CaféScript is built with a simple idea:

> Learning programming should feel like a calm process, not a rushed one.
