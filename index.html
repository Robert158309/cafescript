Rutas de Aprendizaje

Cada persona tiene objetivos diferentes. Algunos quieren crear sitios web, otros desarrollar aplicaciones o trabajar con bases de datos. Estas rutas te ayudarán a seguir un camino claro y aprender de forma progresiva.

Desarrollo Web - Frontend

Si quieres construir sitios y aplicaciones para la web, esta es tu ruta. Comenzamos con los fundamentos y avanzamos hacia tecnologías más complejas.

1. HTML

Construye la estructura de tus páginas web.

2. CSS

Da estilo y personalidad a tus proyectos.

3. Lógica

4. JavaScript
